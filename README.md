# Support-Ticket-System
**Szenario**: Mitarbeiter melden IT-Probleme.

**Tabellen**:
- `users`
- `tickets`
- `categories`
- `comments`

**Funktionen**:
- Ticket erstellen
- Ticketstatus ändern
- Kommentare hinzufügen

**Notification**:
- E-Mail bei neuer Ticketantwort (Statusänderung, Kommentar)
